Keyboard Shortcuts

Open Popup Opens the width control popup
Alt + Shift + W
Cycle Presets Cycles through 50% - 70% - 85% - 100%
Alt + Shift + C
Toggle Default Toggles between current width and 70%
Alt + Shift + D

To customize these shortcuts, go to about:addons, click the gear icon, then select "Manage Extension Shortcuts".

Accessibility

This extension is designed with accessibility in mind:

  • Keyboard Navigation: All controls are accessible via keyboard
  • Screen Reader Support: ARIA labels and live announcements for all changes
  • Reduced Motion: Respects your system's "reduce motion" preference
  • High Contrast: Works with Windows High Contrast mode
  • Focus Indicators: Clear visual focus indicators for keyboard users

About

Claude Chat Width Customizer adjusts the width of the main chat area on claude.ai without affecting the sidebar.

This extension does not collect any user data. Your preferences are stored locally in your browser.
